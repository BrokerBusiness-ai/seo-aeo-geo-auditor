# tools/ — desktop integration

Scripts that turn `seo-aeo-geo-auditor` into a one-double-click app on Windows.

## What's here

| File | Purpose |
|---|---|
| `install_desktop_shortcut.ps1` | One-shot installer: generates `icon.ico` and creates a "SEO Auditor" shortcut on your Desktop. |
| `start.cmd` | The shortcut target. Calls `start.ps1` with the right execution policy. |
| `start.ps1` | Loads API keys from `C:\PYTHON\token\Api_AI.txt` or `.env`, verifies Python, starts `gui.py`, opens browser. |
| `icon.ico` | Generated by the installer (multi-resolution: 16, 32, 48, 64, 128, 256 px). Not committed — re-created on each install. |

## Usage

From the auditor root, run once:

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File tools\install_desktop_shortcut.ps1
```

That places **SEO Auditor.lnk** on your Desktop. Double-click it any time
to launch the audit GUI in your browser. The script:

1. Loads keys from whichever of these exists:
   - `C:\PYTHON\token\Api_AI.txt`
   - `<repo>\.env`
2. Prints provider status (which APIs are wired up)
3. Starts `python gui.py --open` on port 8765
4. Browser opens to `http://127.0.0.1:8765`
5. Window stays open for logs; Ctrl+C stops the server

## Options

```powershell
# Install for all users (writes to Public Desktop, requires admin)
powershell -NoProfile -ExecutionPolicy Bypass -File tools\install_desktop_shortcut.ps1 -AllUsers

# Custom shortcut name
powershell -NoProfile -ExecutionPolicy Bypass -File tools\install_desktop_shortcut.ps1 -ShortcutName "Audit Now"
```

## Uninstall

Just delete the shortcut from your Desktop:

```powershell
Remove-Item "$env:USERPROFILE\Desktop\SEO Auditor.lnk"
```

The repo stays untouched.

## Linux / macOS equivalent

Add this to your shell rc (`~/.bashrc`, `~/.zshrc`):

```bash
alias audit='cd ~/code/seo-aeo-geo-auditor && set -a && source .env 2>/dev/null && set +a && python gui.py --open'
```

Then `audit` from any terminal does the same thing.
